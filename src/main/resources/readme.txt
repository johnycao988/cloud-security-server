# Cloud Config Server
1. set JVM variable CONFIG.SERVER.AUTH.CODE for security validation.
2. set LOGGER.BASE.PATH for log path

# Cloud Config Client 

1. set CONFIG.SERVER.URL (ie:http://10.39.101.226:6069/cloud-config-server/GetConfigFile) for lookup config server.
2. set CONFIG.SERVER.ROOT.PATH (ie. /AppConfig/dev) for locate folder.
